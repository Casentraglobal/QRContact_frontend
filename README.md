# Auth Frontend

Vite + React frontend for the MERN auth demo.

Setup

1. Create a `.env` at the project root with the Google client ID (optional):

```
VITE_GOOGLE_CLIENT_ID=your-google-client-id.apps.googleusercontent.com
```

2. Install deps:

```powershell
cd "d:/QR project/auth-frontend"
npm install
```

3. Run dev:

```powershell
npm run dev
```

Open `http://localhost:3000`.
